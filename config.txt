Конкретный синтаксис для языка L:

Пробельные символы забанены кроме тех мест где они явно указаны.

Функция задаётся следущим образом:

"function <name> n arg_1 arg_2 \dotso arg_n body", где 
name --- имя функции (строка)
n --- количество аргументов, которая принимает функция (10-чное)
arg_i --- собственно сами аргументы (переменные)
body -- тело

body -> "{" + последовательность операторов + "}"

Имена функций должны быть уникальны и не совпадать с ключевыми словами или идентификаторами. 
Одна из функций обязательно задаётся так: "function main 0" --- точка входа в программу.

Литерал -> Число | Строка
Число -> 10-чное | 2-чное
10-чное -> 0([1-9][0-9]* | \eps)
2-чное ->  1(1[0-1]* | \eps)
Строка -> Последовательность символов
Символ -> a | b | \dotso | z

Последовательность obj -> [obj, последовательность obj] | "EoL"

Оператор -> skip | only_if | if | while | bounding | последовательность операторов | вызов функции

skip -> "skip"
only_if -> "only_if" выражение body
if -> "if" выражение body else body
while -> "while" выражение body
bounding -> "bound" (int2 name = 2-чное | int10 name = 10-чное | string name = строка)
вызов функции -> "call" name n arg_1 arg_2 \dotso arg_n

Выражение -> литерал | переменная | вызов функции

Переменная -> name такое, что раньше было связывание с ним
